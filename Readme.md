# OSBB Outages Notification Bot

This project is designed to fetch and notify users about planned and emergency outages in the Chernivtsi Oblast area.

## Features

- Fetch planned and emergency power outage information for specified locations.
- Send notifications via Telegram to a list of predefined chat IDs.
- Configurable for different streets and cities within the Chernivtsi Oblast area.

## Dependencies

- Python 3.x
- Requests
- JSON

Ensure you have Python installed on your system and all required packages are installed:

```
pip install  -r requirements.txt
```
